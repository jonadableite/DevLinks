## Sobre

Este projeto é um conjunto pequeno de arquivos estáticos para estudos e demonstrações: uma página `index.html`, estilos em `style.css`, e scripts em `script.js`. Pode ser usado como ponto de partida leve para aprender conceitos de front-end ou embutir pequenas demos.

## Visualização

Abra [index.html](index.html) no navegador ou execute um servidor local (recomendado).

Exemplos rápidos:

```bash
# com Python 3
python -m http.server 8000

# com npm (instale http-server globalmente)
npx http-server -c-1 . -p 8000
```

Depois, acesse http://localhost:8000

## Estrutura do projeto

- [index.html](index.html) — página principal.
- [style.css](style.css) — estilos.
- [script.js](script.js) — lógica JavaScript.
- [assets/](assets/) — imagens e recursos estáticos.

## Tecnologias

- HTML5

## Instalação / uso

1. Clone o repositório:

```bash
git clone https://github.com/jonadableite/DevLinks.git
cd DevLinks
```

2. Abra `index.html` diretamente ou use um servidor local (veja comandos acima).
